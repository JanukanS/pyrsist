# PYRSIST
## Installation
```bash
uv tool install git+https://github.com/JanukanS/pyrsist
```
## Uninstallation
```
uv tool uninstall pyrsist
```
## Get Started
Create a counter script
```bash
uvx pyrsist init counter.py
```
Get information about the counter script
```bash
uvx pyrsist info counter.py
```

## Using as a Library
Add as a dependency
```bash
uv add git+github.com/JanukanS/pyrsist
```
Call the persist function at the end of the script
